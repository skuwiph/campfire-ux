Buttons

Notes

Embed buttons in a containing div with 'cal' for center-, or 'ral' for right-alignment

<div class="button-bar ral">
    <app-cf-button [type]="Button.Default" (click)="onClick($event)">OK</app-cf-button>
    <app-cf-button [type]="Button.Cancel" (click)="onClick($event)">Cancel</app-cf-button>
</div>

Include icons or additional text using the ng-content block:

<app-cf-button [type]="Button.Danger" [enabled]="true">
Danger (Enabled) <i class="fa-solid fa-circle-exclamation"></i>
</app-cf-button>

If you wish to style any included items, use the :host::ng-deep pattern:

:host::ng-deep button i {{'{'}} margin-left: 0.5rem; {{'}'}}

Remember to import the enum into the component.ts:

Button = CfButtonType;
Primary Secondary Tertiary
Primary Secondary
Default Cancel
Danger (Disabled) Danger (Enabled) Other
